Copyright

Understanding the copyright of the DIYbiosphere project is essential to using and contributing to it. You can review the full legal code of the copyright of the Library (CC0), and the Software (MIT). Alternatively you can read the Terms of Use which explains in simple english the permissions, obligations, and restrictions afforded by these copyrights. In addition, you can review the Contributor Terms, to understand more clearly (in legal terms), the rights granted (MIT) and/or waived (CC0) from your Contributions.
